## üìê Typography

### Font Family
- **Primary:** Inter (Black/ExtraBold weights only)
- **Fallback:** -apple-system, sans-serif

### Type Scale
```
Mega Headline:     text-9xl (128px)  - Hero statements
Large Headline:    text-8xl (96px)   - Page titles
Headline:          text-7xl (72px)   - Section titles
Sub Headline:      text-6xl (60px)   - Sub-sections
Medium Heading:    text-5xl (48px)   - Card titles
Small Heading:     text-3xl (30px)   - Component titles
Body Large:        text-xl  (20px)   - Lead paragraphs
Body:              text-base (16px)  - Main content
Small:             text-xs  (12px)   - Labels, meta
```

### Typography Rules
- **ALL CAPS for headings**
- **Font-black (900 weight) for all headings**
- **Tracking-tight (-0.02em) for large text**
- **Tracking-widest for small text (labels, buttons)**
- **Line height: 0.85-0.9 for mega headlines**

## üìè Layout System

### Spacing
```
Section Padding:   py-24 md:py-40
Card Padding:      p-10 md:p-12
Container Padding: px-6 sm:px-8 lg:px-16
```

## üéØ Design Principles

### 1. Bold Typography
- Massive headlines that demand attention
- Ultra-black font weights only
- All caps for impact
- Tight tracking for large text

### 2. High Contrast
- Pure black backgrounds
- Pure white accents
- Gray tones for hierarchy
- No mid-tones, no gradients

### 3. Geometric Structure
- 4px borders everywhere
- Box shapes, no rounded corners
- Grid-based layouts
- Utilitarian precision

### 4. Functional Minimalism
- Every element has purpose
- No decorative flourishes
- Clean, direct communication
- Rugged simplicity

### 5. Outdoor Brand DNA
- North Face-inspired boldness
- Patagonia-level authenticity
- Arc'teryx precision
- Utilitarian outdoor aesthetic

## üöÄ Component Patterns

### Hero Section
```
- Border-bottom: 4px steel
- Massive headline (text-8xl or text-9xl)
- Accent box for context
- Minimal supporting text
- Bold CTA buttons
```

### Feature Grid
```
- 3-column layout
- Card components with 4px borders
- Hover effect: border turns white
- Numbered or bulleted lists
- Clear hierarchy
```

### Section Header
```
- Accent box label (e.g., "// PROOF")
- Large headline (text-6xl or text-7xl)
- Brief supporting text
- Lots of whitespace
```

### Dividers
```
- Full-width 4px lines
- Steel color default
- White for emphasis
- Generous spacing (my-16)
```
